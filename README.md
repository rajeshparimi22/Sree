# 🌿 Sree Coir Industry Website (React Project)

## 📌 Project Overview

The **Sree Coir Industry Website** is a modern, responsive web application built using **React.js**.
This project represents a company website for a coir manufacturing business, showcasing products, company details, and contact information.

---

## 🚀 Features

* ✅ Responsive Navigation Bar (Navbar)
* ✅ Hero Section with background image and call-to-action
* ✅ About Section (Company information)
* ✅ Products Section (Grid layout with product cards)
* ✅ Contact Section (Company details)
* ✅ Footer Section (Quick links & information)
* ✅ Clean and modern UI design

---

## 🛠️ Technologies Used

* **React.js** (Frontend Library)
* **JavaScript (ES6+)**
* **HTML5**
* **CSS3**
* **VS Code** (Development Environment)


## 🎯 Key Concepts Implemented

### 🔹 Component-Based Architecture

The application is divided into reusable components like Navbar, Hero, Products, etc.

### 🔹 Props & Reusability

Used reusable UI structure for scalability.

### 🔹 CSS Styling

Applied custom CSS for layout, responsiveness, and animations.

### 🔹 Responsive Design

Used flexible layouts (Flexbox & Grid) to support different screen sizes.

## 💡 Challenges Faced & Solutions

| Challenge               | Solution                                      |
| ----------------------- | --------------------------------------------- |
| CSS not applying        | Fixed by proper import and className usage    |
| Component not rendering | Debugged using step-by-step component testing |
| Blank screen issue      | Identified export/import errors               |
| Layout issues           | Used Flexbox & Grid for alignment             |
